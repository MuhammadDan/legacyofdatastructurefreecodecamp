// Question: Create a function that looks through an array arr and returns the first element in it that passes a 'truth test'. This means that given an element x, the 'truth test' is passed if func(x) is true. If no element passes the test, return undefined.

// // Explanation

// Array ke andar se pehla element dhoondhna jo kisi condition ko satisfy karta ho.

// Yeh concept JavaScript mein higher-order functions aur predicate logic ka use karta hai. Challenge kuch is tarah hota hai:

function findElement(arr, func) {
  return arr.find(func);
}

findElement([1, 2, 3, 4], num => num % 2 === 0);
